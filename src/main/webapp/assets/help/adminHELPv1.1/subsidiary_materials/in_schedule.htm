입고예정 등록

물류 > 부자재 관리> 입고예정 등록


수정사항

프로그램 개요
  • 부자재에 대한 입고예정을 등록하는 화면으로 정기 발주 시에만 스케줄이 등록 되었던 것을 이 화면을 통해 임의,정기 구분 없이 등록하여 관리 할 수 있습니다.

    • 기능요약1
  • ① 부자재 발주 시 수량을 입력하면 되며 최소 발주량 보다 적게 입력할 수 없습니다.
  • ② 부자재 발주 시 입고예정일을 입력하면 되며 차후에 입력한 날짜로 계산되어 부자재 스케줄에 표시 됩니다.
  • ③ 발주 시 수량, 입고예정일을 넣고 등록 시 위 화면과 같이 입고예정 데이터가 발생됨과 동시에 발주서가 출력됩니다.
    ※ 같은 업체의 부자재끼리만 발주 가능하며 다른 업체일 경우 경고 창이 표시 됩니다.

    • 기능요약2

    발주서 출력시 상품 비고란을 임의값으로 채워서 출력 할 수 있습니다.
