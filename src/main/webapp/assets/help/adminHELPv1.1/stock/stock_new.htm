재고등록 (물류)

재고관리 전용 > 재고등록 (물류)


수정사항
  • 메뉴 명칭 수정

    • 프로그램 개요
  • 기초상품별로 현재시점에 대한 물류 재고를 확인 및 관리 할 수 있는 화면입니다.

    • 기능요약1
  • 출고가능재고 : 발주가능한 재고
  • 총재고 : 출고가능재고 + 입고존(I) + 반품존(R) + 보류존(H) + 문제존(S) + 폐기존(U) + C급존(Y) + 발주재고
  • 기준값 : 출고가능재고 + 입고존(I) + 제3물류재고
  • 기초상품에 대한 관리(주의,숨김)을 하실경우 해당 체크 박스에 체크 한뒤 등록하기 버튼을 누르시면 됩니다.
  • 자사시스템에서 관리하는 재고를 수정하실경우 재고쪽 입력박스에 수량을 변경하신뒤 등록하기 버튼을 누르시면 됩니다.

    • 기능요약2(이미지)

    기준값과 총재고가 20개 이상차이가 나면 적색으로 표시됩니다. 차이 부분은 확인해서 처리하시기 바랍니다.
