    • 프로그램 개요
  • 검수가 필요한 경우 물류센터에 요청합니다.

    • 기능요약1
  • 검수요청 시 완료 희망일을 입력하여 작성합니다.
  • 물류센터에선 내용 확인 하여 접수 또는 완료처리를 합니다.
  • 검수 진행 시 비용이 발생할 경우 검수완료 예정일과 추가비용을 작성하여 답변합니다.
  • 리스트의 검수방법은 검수완료 예정일이 등록된 답변 중 가장 최근 답변의 검수방법이 노출됩니다.
  • 리스트 하단의 견적추가비용은 총 합계 입니다. (ex. 2페이지의 합계는 1,2페이지 총합입니다.)
