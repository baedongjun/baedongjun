수정사항
  • 기능요약2(이미지) 수정

    • 프로그램 개요
  • 결제완료된 주문내역을 실제 발주로 넘기는 작업을 합니다.

    • 기능요약1
  • 주문번호 PLUS : 해당 주문번호에 추가된 상품이 있음.
  • 주문번호 PLUS PLUS : 해당 상품이 추가된 상품.
  • 주문번호 RE : 재발주내역
  • 주문번호 [삭제] 버튼 : 재발주 내역 삭제 가능
  • 증정품 : 품목명에 파랑색으로 표시

  • [이전발주 결과보기]
    클릭 시 이전 발주 내역을 확인 할 수 있습니다.
    [관리자프로그램관리 > 발주관리> 특별발주 관리]에서 물류다름 체크된 경우 엑셀로 발주내역 다운 받을 수 있습니다.

  • 발주하기(STEP1) 실행 시 발주 대상 목록이 나타납니다.
  • 상단의 검색 조건은 특별발주와 특별발주제외한 내용을 조회 가능하며 숫자는 건수를 나타냅니다.
  • 특별 발주인 경우 포장 항목에서 발주 묶음을 그룹핑 하면 그룹핑 대로 송장이 나눠집니다.
  • 발주완료 처리하면 물류에 발주가 들어가지 않고 완료되며 발주취소처리 하면 발주가 다음발주로 미뤄집니다.
  • 발주 내용 변경 및 발주가 가능합니다.
  • [물류로 발주하기 (STEP2)]로 하면 실제 발주가 이뤄집니다. 그 후 발주 결과 내역을 조회 할 수 있습니다.
