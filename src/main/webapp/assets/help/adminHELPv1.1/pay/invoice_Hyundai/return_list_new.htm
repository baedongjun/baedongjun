물류 > 회수관리(온라인) > 회수완료관리

수정사항
  • 기능요약2(이미지) 추가

    • 프로그램 개요
  • 회수 요청된 내역을 회수 확인 후 완료처리를 합니다.

    • 기능요약1
  • 해당 입고처에서는 회수처리 부분에 있는 [처리] 버튼을 클릭하여 내용을 입력 후 처리 완료를 합니다.
  • 요청내역이 없을 경우 하단의 [회수요청없이 회수된 내역 등록]을 통하여 회수내역을 입력 합니다.
  • [제외]로 처리 하면 회수 요청이 취소 됩니다.
  • 상품의 왼쪽 내역은 회수요청 내역이며 오른쪽은 회수된 내역입니다.

    • 기능요약2(이미지)
