사용후기 관리

사이트 관리 > 사용후기 관리

프로그램 개요
  • 상품의 사용후기를 관리합니다.

    • 기능요약1
  • 쁘띠엘린 스토어, 브랜드 사이트에서 작성한 후기를 확인 가능합니다.(현재 모바일 스토어는 후기 작성이 불가능)
  • 후기 작성 시 쿠폰 지급됩니다. (월 5회 이하로 제한, 일반 후기 : 500원 할인 쿠폰, 포토후기: 2000원 할인 쿠폰 지급)
  • [적용하기] : Best Review 등록 여부를 적용합니다.
  • [삭제하기] : 해당 후기로 쿠폰 발급 여부, 발급된 쿠폰의 사용 여부를 체크를 합니다. 그 후 발급된 쿠폰이 사용되지 않았다면 해당 쿠폰을 삭제 및 해당 후기를 삭제 합니다.
  • [QandA 관리로 이동] : 해당 후기를 QandA로 이동 후 후기에서 삭제 합니다.
  • [카테고리 이동] : 구분 항목에서 선택 된 카테고리로 변경합니다.
