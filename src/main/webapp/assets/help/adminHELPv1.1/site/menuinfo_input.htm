수정사항

프로그램 개요
  • 쁘띠엘린 스토어 사이트의 상단 마우스 오버 시 확장되는 메뉴를 관리합니다.

    • 기능요약1
  • 쁘띠엘린 각 카테고리의 추천브랜드와 배너를 관리합니다..
  • 각 카테고리별 추천브랜드는 6개까지 설정 가능합니다.
  • 배너 이미지와 배너 클릭 시 이동될 URL 링크를 설정 가능합니다.
  • 배포하기 버튼 클릭 시 현재 등록한 상태로 메뉴가 변경 됩니다.
    • 기능요약2(이미지)


    메뉴 화면이 이상 없는지 확인 하고 신중하게 배포해 주시기 바랍니다.
