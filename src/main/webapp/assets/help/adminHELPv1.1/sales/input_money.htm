정산금액 입력

원가/공헌이익 전용 > 정산금액 입력


수정사항
  • 기능요약2(이미지) 추가

    • 프로그램 개요
  • 각월별 해당 몰별로 정산금액을 등록 할 수 있는 화면입니다.

    • 기능요약1
  • 등록하는 정산금액은 각 타사몰에서 적용되는 수수료를 제거한 실제 정산금액을 입력합니다.
  • 환불 / 취소의 경우 계산에 넣지 않습니다.(정산 상에 - 값)
  • 업로드시 반드시 CSV 파일로 업로드 하십시오. (형태: 주문번호, 정산금액)
  • 업로드시 해당몰에 해당월에 자료가 있으면 덮어씌워집니다.

    • 기능요약2(이미지)

    자료 업로드후 하단에 있는 현황을 통해 금액을 확인후에 "정산금액 처리하기" 버튼을 누르시면 최종 반영됩니다.
