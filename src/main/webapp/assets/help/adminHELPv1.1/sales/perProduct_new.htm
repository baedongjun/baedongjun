    • 프로그램 개요
  • 해당년도의 각분기별로 기초상품에 대한 RRP,실매출,원가,공헌이익,공헌이익률을 확인 할 수 있는 화면입니다.

    • 기능요약1
  • 공헌이익 = 실매출 - 원가 - 수수료
  • 공헌이익률 = 공헌이익 / RRP
  • RRP = 각 기초상품에 대한 RRP * 판매 수량
  • 실매출 = 정산금액 * 판매수량
  • 원가 = 각 기초상품에 대한 원가 * 판매 수량
  • RRP가 0인 제품은 계산에서 제외됩니다.(월별 증정내역 참조)
  • 목록을 엑셀파일로 다운로드할 수 있습니다.

    • 기능요약2(이미지)

  • 각 브랜드별로 해당월에 공헌이익,공헌이익률이 가장 큰 기초상품은 빨강, 가장 적은 기초상품은 파랑색으로 표시 됩니다.
