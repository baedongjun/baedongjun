부서관리

관리자 프로그램관리 > 인원관리 > 부서관리

수정사항

프로그램 개요
  • 관리자프로그램내에서 사용하는 부서를 등록하거나 관리하는 화면입니다.

    • 기능요약1
  • 부서명에 내용을 모두 지우시고 등록하시면 목록이 삭제됩니다.
  • 부서명에 "BM"이 들어간 경우 "기초상품"에서 단가(cost)를 확인할 수 있습니다.
  • 담당몰은 MD팀에서 각 담당하는 몰을 선택하면 됩니다.

    • 기능요약2(이미지)

    부서관리는 관리자프로그램내에서 사용하는 부서를 등록하거나 관리하는 화면으로서 업무혁신팀에서 관리하고 등록합니다.
