  • 안건 분류 배분(경영기획 팀원 중 1인 지정 : 한정빈)
  • 배분된 안건은 경영기획, 전략기획팀 답변 후 팀장 확인
  • 팀장 확인된 답변 또는 확인요청 안건은 취합 후 대표이사 전달
  • 대표이사 안건에 대한 답변 확인 후 변경사항 전달 및 주간 경영회의 상정 여부 결정
  • (답변에 대한 변경사항은 다시 해당 담당자에게 전달)
  • 경영회의 상정 안건은 경영진 검토, 실행 가능여부 확인 후 보상
  • 그 외 안건은 대표이사 확인 후 보상
  • 보상지급은 월1회
