수정사항

프로그램 개요
  • 중요 업무공지내용을 공유하는 게시판입니다.

    • 기능요약1
  • 긴급한 메세지는 '공지'로 지정하여 등록합니다.
  • 메일로 전달된 내용은 담당자가 등록함 (임시담당자 : 황은진과장)
  • 고객만족팀에서 요청건에 대해 엑셀다운을 받은후 물류측에 보내서 처리 합니다.
  • 등록 시 양식에 맞춰 등록합니다.
    *등록양식
    [제목] [브랜드명] 주요키워드 / 담당자
    ex) [킨더스펠] 라지블랭킷 라벨 부착위치 변경 / BM2팀 이승은

    [내용] 메일전달 내용 및 담당자와 대화내용 입력

    • 기능요약2(이미지)

    업무내용만 공유, 임의로 삭제하거나 변경하지 말 것 (내용에 상관없는 오타수정, 어휘수정 등 제외)
