월별 매출

국내영업 발주 > 매출 통계 관리 > 월별 매출

프로그램 개요
  • 업체 선택 하면 해당 브랜드의 상품별 내역이 조회 가능하며 매출 변동내역 입력 및 조회 할 수 있습니다.

    • 기능요약1
  • 매출 변동내역의 업체는 그룹으로 묶인 업체가 조회됩니다.(그룹 등록 안된 경우 개별로 조회됨)
  • 금액은 발주 시 RRP(판매가) + 변동내역 입니다.

    • 기능요약2(이미지)
  • 리스트에 붉은 부분은 변동내역이 등록된 업체 입니다.


  • 리스트의 업체를 클릭할 경우 해당 브랜드의 구매 내역이 조회 가능합니다.
    ①그룹으로 묶인 업체인 경우 그룹에 해당하는 업체의 구매내역이 전부 조회 됩니다.
    ②변동내역 등록 시 업체 리스트는 [국내영업 발주 > 매출 통계관리 > 매출그룹관리]에서 그룹으로 등록되었을 경우 아래와 같이 보이며, 그룹이 아닌 경우 해당 업체만 나타납니다.
    ③변동내역이 등록되어 있을 경우 왼쪽의 구매 내역에서 붉은색으로 표시 됩니다.

    • [국내영업 발주 > 매출 통계 관리 > 월별매출(구)]와 동일 합니다.
