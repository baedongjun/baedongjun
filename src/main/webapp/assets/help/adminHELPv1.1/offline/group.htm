기본 상품관리

국내영업 발주 > 기본 상품관리

프로그램 개요
  • 국내영업 발주 및 통계에서 사용할 상품을 관리 합니다.

    • 기능요약1
  • 그룹명 선택 후 삭제 시 해당 그룹에 등록된 단품/팩키지가 전부 삭제 됩니다.
  • 할당되지 않은 상품에서 추가 하거나 등록된 단품/팩키지를 삭제 후 등록을 하면 적용됩니다.

    • 기능요약2(이미지)
  • 브랜드 선택 후 그룹을 추가 하거나 수정, 삭제를 할 수 있습니다.
  • 그룹에 해당 하는 상품은 오른쪽 단품/팩키지에서 선택하여 추가 후 등록 하면 해당 상품을 발주 할 수 있습니다.

    • 위와 같이 각 업무에서 상품그룹 관리를 하는 곳이 [MD전용 > 상품 그룹 관리], [재고관리 전용 > 상품 그룹 관리], [국내영업 발주 > 기본 상품관리] 입니다.
