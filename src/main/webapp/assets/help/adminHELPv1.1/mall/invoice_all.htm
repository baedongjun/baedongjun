몰 발주서 등록

타사몰 발주 관리 > 몰 발주서 등록

수정사항

프로그램 개요
  • 몰에서 받은 발주 내역을 등록하여 전체리스트로 넘깁니다.

    • 기능요약1
  • 발주 내역을 등록 후 구매내역, 구매자, 수취인, 금액 등을 꼭 확인 한 다음에 전체 리스트로 넣습니다.
  • [같은 주문번호에 추가하기] : 추가할 내역을 체크 후 추가하기 하면 해당 주문번호로 추가됩니다.
  • 검색조건에 있는 몰은 발주내역이 등록된 타사몰을 표시 합니다.
  • NO에 붉은색으로 표시된 내용은 이미 발주된 내역임을 나타냅니다. (예약발송이나 상품문제 발생 등)
  • 할인율이 40% 이상 50% 미만인 경우 주황색으로 표시 되며 50% 이상인 경우 붉은색으로 표시되니 꼭 할인율을 확인해야 합니다.

    • 기능요약2(이미지)

    발주 내역 등록 후 해당 내용을 꼼꼼히 확인 후 전체리스트로 넣습니다.
