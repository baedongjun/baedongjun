정산내역

원가/공헌이익 전용 > B2B 공헌이익 > 정산내역


수정사항

프로그램 개요
  • 해당월에 대한 각 부서의 브랜드별로 정산내역을 확인 할 수 있는 화면입니다.

    • 기능요약1
  • 원가 = 각 기초상품에 대한 원가 * 판매 수량
  • RRP = 각 기초상품에 대한 RRP * 판매 수량
  • 정산금액 = 정산금액 입력(엑셀입력값) / 1.1 (=공급가)
  • 수수료 = 통합수수료관리(엑셀입력값)
  • 반품 COST = 반품된 각 기초상품에 대한 원가 * 반품 수량
  • RRP = 반품된 각 기초상품에 대한 RRP * 반품 수량
  • 목록을 엑셀파일로 다운로드할 수 있습니다.

    • 기능요약2(이미지)

    각 부서 별로 정산금액이 가장 큰 브랜드는 빨강, 가장 적은 브랜드는 파랑색으로 표시 됩니다.
