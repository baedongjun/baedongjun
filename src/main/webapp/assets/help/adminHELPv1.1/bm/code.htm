자재코드 그룹관리

BM 전용 >자재코드 그룹관리

수정사항

프로그램 개요
  • 제품제작등에 사용되는 원/부자재를 구분하는 그룹코드를 등록하여 자재성격별로 구분하여 관리 합니다.

    • 기능요약1
  • 원/부자재 성격에 따라 코드를 생성합니다.
  • 해당 코드에 대한 세부 설명을 기록합니다.
  • 그룹코드를 생성하지 않아도 실제 원/부자재 등록시 생성한 코드를 기준으로 생성되지 않은 코드들도 표시됩니다.
  • 해당 코드를 클릭하면 자재 기본관리로 연동되어 세부 원/부자재 정보를 확인할 수 있습니다.

    • 기능요약2(이미지)

    그룹코드를 등록하지 않아도 원/부자재 등록시 생성한 코드를 분류하여 나타나는 항목도 있습니다.
