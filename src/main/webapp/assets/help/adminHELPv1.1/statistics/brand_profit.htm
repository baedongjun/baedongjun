기타 참고 > 브랜드별 이익 산출


수정사항

프로그램 개요
  • 각 브랜드별로 해당년도의 월별 이익 산출 정보를 등록하고 확인하는 화면입니다.

    • 기능요약1
  • 엑셀양식을 다운받아 엑셀작업후 엑셀 일괄등록을 통하여 이익 산출 정보를 등록 할 수 있습니다.
  • 엑셀 업로드시 파일 확장자를 확인하십시오(.xls만 가능)
  • 총공헌이익 = 실매출 - ( 총원가 + 판매수수료 + 마케팅비 + 창고비 + 택배비)
  • 이익율 = 총공헌이익 / RRP매출
  • 각 브랜드에서 붉은색, 파란색 금액은 각 항목별(RRP매출, 총원가, 실매출등)의 년중 최대값, 최소값을 나타냅니다
  • 또한 누적 TTL 항목에서의 최대, 최소값은 브랜드중 최대, 최소인 항목으로 표시됩니다.
  • 목록을 엑셀파일로 다운로드할 수 있습니다.

    • 기능요약2(이미지)
