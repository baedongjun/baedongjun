기타 참고 > 재구매 > 재구매율


수정사항
  • 하단 내용 수정

    • 프로그램 개요
  • 선택한 브랜드 or 카테고리 or 기초상품 별로 재구매 정보를 확인 할 수 있는 화면 입니다.

    • 기능요약1
  • 자사 회원 기준으로 계산합니다.
  • 재구매율 계산을 위한 데이터 생성은 서버에 무리가 가므로 재구매 기본 데이터 생성일자 기준으로 2일 후에 재 생성이 가능합니다.
    - [재구매 기본 데이타 생성일자] 기준으로 2일이 지나면 데이터 생성을 위한 버튼이 생깁니다.
    - 재구매율 계산은 위해 데이터를 생성하여 계산하기 때문에 데이터 생성시점과 현재시점의 차이가 있어 조회된 내용이 약간 차이가 발생 할 수 있습니다.
    - 재구매율 기본 데이타 생성 시 다소 시간이 걸립니다.
  • 재구매율 조회는 최대 3개 파트까지 조회가 가능합니다.
  • 비교 내용 선택 시 드래그나 Ctrl+클릭으로 여러 건으로 선택 가능하며 대상에 해당하는 하단부분의 선택버튼을 클릭하여 비교내용을 세팅 합니다.

    • 기능요약2

    재구매율 대상 구매 내역은 최근 6개월 기준입니다. (현재 주문 내역이 6개월치만 존재합니다.)
    단품 검색일 경우 같은 장바구니에 있는 경우도 재구매로 봅니다.
    검색결과는 해당 상품의 구매시점을 기준으로 같이 구매했거나 이후에 구매한 내용만으로 구성됩니다.
